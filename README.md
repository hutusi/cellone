cellone -- a toolkit for audit
==============================

Getting Started
---------------
+ Clone the repo, `git clone git@github.com:hutusi/cellone.git`
+ Build with Visual studio 2010 
+ dependency: ClosedXML (for .Net Framework 3.5), download from: http://closedxml.codeplex.com/
